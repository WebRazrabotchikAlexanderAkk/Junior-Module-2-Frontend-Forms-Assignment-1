# 🚀 Приложение регистрации пользователя

Простое и понятное приложение для регистрации нового пользователя, созданное с использованием **React** и **Vite**.

## 📋 Задача

Создать форму регистрации с тремя полями:

- **Email**
- **Пароль**
- **Повтор пароля**

с поддержкой:

- ✅ Валидации полей
- ✅ Отображения ошибок
- ✅ Блокировки кнопки при невалидной форме
- ✅ Автофокуса на кнопке при валидной форме
- ✅ Вывода данных в консоль после отправки
- 🎨 Свободный дизайн — на усмотрение разработчика

## 🧪 Вторая часть
